## 프로젝트 개요 (Project Overview)
- **목적** : Spring MVC 구조와 JDBC를 활용한 기본적인 이커머스 시트템 구현
- **개발 기간** : 2023.10 ~ 2023.12

## 기술 스택
**Backend**
- Java 11
- Framework : SpringFramework 5.3.10
- Database : MySQL
- ORM/Data Access : Spring JDBC (JdbcTemplate)
- Security : Spring Security

**Frontend**
- JSP/JSTL
- Bootstrap 5.3.2
- JavaScript

## 주요 구현 내용
**회원 관리**
- BCryptPasswordEncoder를 통한 비밀번호 암호화
- 세션을 통한 로그인 상태 유지 및 권한별 접근 제어

**상품 서비스**
- 상품 목록 조회 및 검색 : `LIKE`연산자를 통한 검색기능 구현
- 상품 등록
- 리뷰 시스템

**장바구니**
- 장바구니 관련 시스템 구현
